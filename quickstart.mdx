## Get Started in 5 Minutes

This guide will help you make your first API request and start tracking social media videos.

<Steps>
  <Step title="Create Your Account">
    Sign up at [contentstats.io/auth/signup](https://contentstats.io/auth/signup)
    
    You'll receive **$5.00 free credit** (~333 free snapshots) to get started.
  </Step>

  <Step title="Generate an API Key">
    1. Go to your [Dashboard](https://contentstats.io/dashboard)
    2. Navigate to **API Keys**
    3. Click **"Create New Key"**
    4. Copy your key (it starts with `cs_live_...`)
    
    <Warning>
      Your API key is shown **only once**. Store it securely and never commit it to version control.
    </Warning>
  </Step>

  <Step title="Track Your First Video">
    Use the `/api/v1/videos/track` endpoint to start monitoring:

    <CodeGroup>
      ```bash cURL
      curl -X POST https://contentstats.io/api/v1/videos/track \
        -H "Content-Type: application/json" \
        -H "X-API-Key: cs_live_YOUR_KEY" \
        -d '{
          "video_link": "https://www.tiktok.com/@user/video/123",
          "duration_days": 7
        }'
      ```

      ```javascript JavaScript
      const response = await fetch('https://contentstats.io/api/v1/videos/track', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-API-Key': 'cs_live_YOUR_KEY'
        },
        body: JSON.stringify({
          video_link: 'https://www.tiktok.com/@user/video/123',
          duration_days: 7
        })
      });

      const data = await response.json();
      console.log(data);
      ```

      ```python Python
      import requests

      response = requests.post(
          'https://contentstats.io/api/v1/videos/track',
          headers={
              'Content-Type': 'application/json',
              'X-API-Key': 'cs_live_YOUR_KEY'
          },
          json={
              'video_link': 'https://www.tiktok.com/@user/video/123',
              'duration_days': 7
          }
      )

      data = response.json()
      print(data)
      ```

      ```php PHP
      <?php
      $curl = curl_init();

      curl_setopt_array($curl, [
          CURLOPT_URL => 'https://contentstats.io/api/v1/videos/track',
          CURLOPT_RETURNTRANSFER => true,
          CURLOPT_POST => true,
          CURLOPT_HTTPHEADER => [
              'Content-Type: application/json',
              'X-API-Key: cs_live_YOUR_KEY'
          ],
          CURLOPT_POSTFIELDS => json_encode([
              'video_link' => 'https://www.tiktok.com/@user/video/123',
              'duration_days' => 7
          ])
      ]);

      $response = curl_exec($curl);
      $data = json_decode($response, true);
      print_r($data);
      ?>
      ```
    </CodeGroup>

    <CodeGroup>
      ```bash cURL
      curl https://contentstats.io/api/v1/videos/cm5abc123def \
      ```

## What Happens Next?

Once tracking starts:

1. **Hourly Snapshots**: We collect metrics every hour for the duration you specified
2. **Bot Analysis**: AI analyzes engagement patterns for authenticity
3. **Automatic Billing**: Each snapshot costs $0.015, deducted from your balance
4. **Email Updates**: Get notified when tracking completes or if your balance is low

## Cost Calculation

Understanding your costs:

```
Cost = Duration (days) × 24 hours × $0.015 per snapshot

Example for 7 days:
7 × 24 × $0.015 = $2.52
```

<Info>
  Your **$5 free credit** covers about 333 snapshots, or ~14 days of tracking a single video.
</Info>

## Common Questions

<AccordionGroup>
  <Accordion title="How soon will I get my first snapshot?">
    The first snapshot is captured within the first hour of tracking. You'll receive hourly updates after that.
  </Accordion>
  
  <Accordion title="Can I stop tracking early?">
    Yes! Use the [stop endpoint](/api-reference/videos/stop-video) to end monitoring at any time. You'll only pay for snapshots already collected.
  </Accordion>
  
  <Accordion title="What if my balance runs out?">
    Tracking will automatically pause. Add more credit to resume. Your data is saved and tracking continues from where it left off.
  </Accordion>
  
  <Accordion title="Can I track multiple videos?">
    Absolutely! Track as many videos as you want across all platforms simultaneously.
  </Accordion>
</AccordionGroup>
