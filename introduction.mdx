## Quick Start

Get started in 3 simple steps:

<Steps>
  <Step title="Create Account">
    [Sign up](https://contentstats.io/auth/signup) and get $5 free credit
  </Step>
  <Step title="Generate API Key">
    Create an API key from your [dashboard](https://contentstats.io/dashboard/api-keys)
  </Step>
  <Step title="Make Your First Request">
    ```bash
    curl -X POST https://contentstats.io/api/v1/videos/track \
      -H "Content-Type: application/json" \
      -H "X-API-Key: cs_live_YOUR_KEY" \
      -d '{
        "video_link": "https://www.tiktok.com/@user/video/123",
        "duration_days": 7
      }'
    ```
  </Step>
</Steps>
