## When to Contact Support

### Technical Issues

Contact us if you're experiencing:
- API errors not documented
- Unexpected behavior
- Integration problems
- Performance issues

**Please include:**
- Error messages (full text)
- Request/response examples
- Video IDs or tracking IDs
- Timestamp of the issue

### Billing Questions

Reach out for:
- Refund requests
- Invoice questions
- Payment issues
- Pricing information

### Feature Requests

We love hearing your ideas! Request features for:
- New platforms (e.g., Snapchat, LinkedIn)
- New metrics
- Webhook events
- API improvements

## Report a Bug

Found a bug? Help us fix it quickly:

<Steps>
  <Step title="Verify It's a Bug">
    - Check docs to confirm expected behavior
    - Test with different inputs
    - Review [API Reference](/api-reference/introduction)
  </Step>
  
  <Step title="Gather Information">
    **What we need:**
    - Steps to reproduce
    - Expected vs. actual behavior
    - Error messages
    - Request/response examples
    - Platform (TikTok, YouTube, Instagram, Twitter)
  </Step>
  
  <Step title="Submit Report">
    Email: **me@siya.digital**
    
    Subject: `[BUG] Brief description`
    
    Include all information from Step 2
  </Step>
</Steps>
