## When to Contact Support

### Technical Issues

Contact us if you're experiencing:
- API errors not documented
- Unexpected behavior
- Integration problems
- Performance issues

**Include:**
- Error messages (full text)
- Request/response examples
- Video IDs or tracking IDs
- Timestamp of the issue

### Billing Questions

Reach out for:
- Refund requests
- Invoice questions
- Payment issues
- Enterprise pricing

### Feature Requests

We love hearing your ideas! Request features for:
- New platforms (e.g., Snapchat, LinkedIn)
- New metrics
- Webhook events
- API improvements

## Report a Bug

Found a bug? Help us fix it quickly:

<Steps>
  <Step title="Verify It's a Bug">
    - Check docs to confirm expected behavior
    - Test with different inputs
    - Check if it's a known issue
  </Step>
  
  <Step title="Gather Information">
    **What we need:**
    - Steps to reproduce
    - Expected vs. actual behavior
    - Error messages
    - Request/response examples
    - Platform (TikTok, YouTube, etc.)
  </Step>
  
  <Step title="Submit Report">
    Email: **bugs@contentstats.io**
    
    Subject: `[BUG] Brief description`
    
    Include all information from Step 2
  </Step>
</Steps>

3. Include: timestamp, endpoint, error details

## Security Issues

Found a security vulnerability?

**DO NOT** create a public issue or post in Discord.

Email: **security@contentstats.io** with:
- Vulnerability description
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

We take security seriously and respond within 24 hours.
